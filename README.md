# Algorithms

Репозиторий курса `Алгоритмы и структуры данных`

### Установка

```shell
pip install pipenv
```

```
pipenv shell
```

После установки pipenv в левом нижнем углу при использовании `vscode` следует выбрать версию интерпретатора, привязанную к проекту
